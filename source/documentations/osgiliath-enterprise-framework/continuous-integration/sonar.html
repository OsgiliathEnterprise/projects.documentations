Sonar

In order to add sonar analysis on your project, you've to add a profile on your $MAVEN_HOME/conf/settings.xml: ``` xml sonarqube jdbc:mysql://localhost:3306/sonardb?useUnicode=true&characterEncoding=utf8 com.mysql.jdbc.Driver sonarusr sonarpwd http://localhost:9000/ /var/lib/jenkins/.sonar/sonar-secret.txt ``` That's it! you just have to run mvn sonar:sonar to run a sonar analysis on the project
